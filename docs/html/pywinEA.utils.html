pywinEA.utils.fast_non_dominated_sort module¶

pywinEA.utils.fast_non_dominated_sort.dominance(solution_1, solution_2)¶

Function that analyze solutions dominance.

Parameters

  • solution_1 – Solution

  • solution_2 – Solution

:return int

If solution_1 dominates solution_2 -> return 1

:return -1

If solution_2 dominates solution_1 -> return -1

:return 0

If neither solution dominates the other -> return 0
